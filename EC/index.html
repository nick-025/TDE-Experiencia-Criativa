  • Projeto I
    Neste projeto, nós criamos um jogo usando a plataforma Construct 2. Foi o nosso primeiro
    contato com criação de jogos. Esta é uma plataforma de muito alto nível, e simples de usar.
    fizemos o trabalho em equipe de 5 pessoas, e fomos os vencedores do melhor jogo criado entre a turma.
    Nosso jogo era o Dark Hut. Os integrantes do grupo eram: Carlos Henrique dos Santos, Gabriel Scholze,
    Matheus Muller, Nicholas Davi e Vinicius Emery Flores.


  • Projeto II
    Neste projeto, nós criamos uma interface multimídia desenvolvida em Java no ambiente
    Processing 3. Este projeto contava com recursos audiovisuais e de interação com o usuário.


  • Projeto III
    Neste projeto, nós criamos uma interface interativa desenvolvida em C# no ambiente
    profissional Visual Studio Enterprise 2019. Nós utilizamos o WinForms para criar janelas
    e exibir conteúdos que desenvolvemos em aula neste projeto, contando com os recursos
    nativos da ferramenta.
